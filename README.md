🎓 Academic Insight Engine

A professional Java 8 based Student Performance Analysis and Report Generation System designed to evaluate academic records, calculate grades, analyze performance metrics, and generate detailed student reports using modern Java programming concepts.

📌 Project Overview

Academic Insight Engine is an advanced version of a traditional Student Grade Calculator. Instead of performing only basic calculations, the system applies performance analysis, grading rules, division classification, subject-wise evaluation, and report generation using Java 8 features and Object-Oriented Programming principles.

This project demonstrates practical implementation of:

Core Java
Java 8 Streams API
Lambda Expressions
